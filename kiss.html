Topologia

É um sistema que demonstra como os hosts estão conectados na rede.

O que é topologia de redes?

Existem alguns modelos que pode sistematizar a conexão de cada host.

Existem dois tipos de topologia de rede:

Topologia Física: Demonstra de fato como os hosts estão dispostos no ambiente físico.

Topologia Lógica: Demonstra a forma como os dados são transmitidos na rede.

Benefícios e desvantagens de cada Topologia:

Barramento: Todos os hosts estão conectados ao mesmo meio físico, embora os dados não circulem dentro de cada um de nós

Benefícios:

Utilizam poucos cabos; sua instalação é simples, seus meios físicos são mais baratos.

Simples e confiável.

Topologia simples e confiável.

Permite a ampliação da rede sem complicações.

Desvantagem:

Comunicação entre os hosts pode ficar lenta.

Problemas difíceis de isolar.

Quando se tem um problema, o mesmo é difícil de identificação.

Se tiver uma falha no cabo de rede, a rede de sua empresa não irá funcionar.

Estrela

É a topologia mais usada, ela usa o meio físico de par trançado com um centralizador de rede, que se encarrega de fazer a comunicação dos hosts.

Benefícios

Facilita a instalação e configuração dos computadores da rede.

A gerência dos equipamentos é centralizado.

Quando um computador está com falha, não paralisa a rede.

Desvantagem:

Se o centralizador da rede apresentar falha, ele paralisa a rede.

Anel:

Estes hosts são conectados em série, que se caracteriza um anel, estes são propagados em uma única direção até chegar em seu destino.

Benefícios:

Os hosts conseguem acessar a rede uniformemente.

O crescimento dos hosts não altera o desempenho da rede.

Desvantagens:

Se un host falhar, impactará em toda rede; difícil isolar os hosts.

Malha:

É uma das mais utilizadas, pois facilita a instalação e configuração dos hosts.

Benefícios:

Maior abundância e confiabilidade; agilidade para reconhecer falhas na rede.

Desvantagem:

Alto custo de implantar.

Ipv4, Máscara de rede e Getaway

introdução:

Para obter a comunicação entre os hosts da rede, precisa da parte física para que a rede tenha uma comunicação entre os hosts, tem que ter a configuração lógica.

Ipv4

Tecnologia que permite que nossos hosts trafeguem na internet, e todo host on line tem um código que permite enviar e receber informações de outors hosts conectados na rede.

O ipv4 é formado por 04 blocos de 8 bits, os endereços que iniciam 192 168 com 10 ou com 172.16 até 172.31, são poupados para redes locais.

O ipv4 transfere endereços de protocolos 32 bits. Sustenta mais ou menos 4,29 bilhões de IPS pelo mundo.

O ipv4 suporta 4,29 bilhões de IPS pelo mundo todo.

Máscara de rede

É um bairro que possui várias residências.Os ips que utilizam uma determinada classe, pertence aquela classe.Os ips possuem 4 octetos de binários que tem 32 bits.

Classe A: 10.0.0.0 até 10.225.255.255(255.0.0.0)-Utiliza apenas o 1º octeto do IP que fica exclusivo para a máscara de rede e o resto para os hosts.

Classe B: 172.16.0.0 até 172.31.255.255(255.255.0.0)-Utiliza os dois 1° octetos do IP que fica exclusivo para a máscara de rede e o resto

Classe C: 192.168.0.0 até 192.168.0.0(255.255.255.0)-Utiliza os três 1º octetos do IP que fica exclusivo para a máscara de rede e o resto para os hosts.

O que é Getaway?

É um host de rede no qual se tem duas redes. Podemos traduzir como "portão ou portal", é assim que funciona, é um portão que libera o acesso ou não para outras redes.

O que essa tecnologia libera?

A passagem dos dados entre hosts, no qual pode ser instalados políticas de acesso, que vêem a segurança dos dados do computador até a internet.

O que o gateway libera?

O acesso pela internet, por exemplo tendo 04 computadores, com 04 IPS diferentes que estão na mesma máscara de rede, para não ter um modem ou outro dispositivo que conecte a internet em cada computador tem um getaway que compartilha a internet para outra rede em todos os computadores.

Exemplos:

Computador A: IP 192.168.1.100, máscara de rede: 255.255.255.0, getaway: 192.168.1.1

Computador B: IP 192.168.1.158, máscara de rede: 255.255.255.0, getaway: 192.168.1.1

Computador C: IP 192.168.1.200, máscara de rede: 255.255.255.0, getaway: 192.168.1.1

Computador D: IP 192.168.1.254, máscara de rede: 255.255.255.0, getaway: 192.168.1.1

Configurando um cliente na rede

Existem duas formas de fazer, o dinâmico onde só insere o cabo de rede e o protocolo DHCP que inclui IP, Gateway, DNS e Máscara de Rede, e a outra forma é a estática.
